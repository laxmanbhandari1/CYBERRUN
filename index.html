CHALLENGES
Challenges & How We Solved Them
MAP
Road tiles used fixed positions. Tiles teleported visibly as player moved.
→
✓
Tiles recycle only when 20+ units behind player. Exact loop maths. Seamless city.
QIZ
Quiz popup on every hit frustrated players. Children clicked randomly to exit.
→
✓
2.5s sliding tip card replaces forced quiz. Game continues. Tip read passively.
DB
Multiple runs stacked — same player appeared multiple times on leaderboard.
→
✓
Fetch existing row first. Update ONLY if new score is higher. One row per player.
WIN
Portrait game appeared as tiny box on landscape desktop screen.
→
✓
stretch/mode=viewport in project.godot. Window auto-resizes to portrait ratio.

About the Game

Learning disguised as play

CyberDefend Runner is a 3D cyberpunk endless runner where the city is under digital attack. You are the last line of defence. Run through neon-lit streets, dodge real cyber threats — Hackers, Viruses, Phishing attacks, Ransomware — and learn how to defeat them in real life.

Built with Godot 4 and powered by a real-time Supabase backend, every run is tracked globally. Your rank reflects your knowledge — not just your reflexes. From Script Kiddie to CISO, the journey is real.
